Usage

Add data-aeos-shell="sidebar" to your layout wrapper. Default: 56px icon-only rail. Add .sidebar-expanded to host for 240px labeled sidebar. JS-togglable.

Responsive

Below 768px the sidebar collapses and hides. Add .mobile-open to host to overlay it at full 240px width as a drawer. Ideal for mobile navigation.

Usage

Add data-aeos-shell="topnav" to your layout wrapper. Supports brand slot, nav links, and an actions slot on the right. Optional sub-bar for breadcrumbs.

Responsive

Below 640px the nav links wrap to a second row below the brand and actions. The topbar gains height automatically. No JS required for the responsive collapse.

Usage

Add data-aeos-shell="floating" to your layout wrapper. The sidebar is a 56px floating pill with border-radius: 14px, backdrop blur, and a drop shadow. Content panel also gets rounded corners.

Responsive

Below 640px the sidebar shrinks to 44px and reduces padding. Expand to 200px labeled mode with .sidebar-expanded on the host element. Best suited for dense dashboard layouts.

Usage

Add data-aeos-shell="command" to your wrapper. Three-column: .aeos-shell-left (160px nav), .aeos-shell-main (flex), .aeos-shell-right (240px context). Use .aeos-shell-widget for right-panel sections.

Responsive

At 1200px the right panel hides (two-column). At 768px the left panel hides (single-column). Inspired by VS Code and Figma's workspace layout patterns for power users.
