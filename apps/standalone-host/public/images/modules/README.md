# Module Thumbnails

This directory contains thumbnail images for marketplace modules.

## Image Requirements
- **Size:** 400x300px (or similar 4:3 ratio)
- **Format:** PNG or JPG
- **Naming:** `{module-code}.png` (e.g., `hrm.png`, `crm.png`)

## Modules
Place thumbnail images for the following modules:

- `hrm.png` - Human Resource Management
- `crm.png` - Customer Relationship Management
- `finance.png` - Finance & Accounting
- `project.png` - Project Management
- `pos.png` - Point of Sale
- `scm.png` - Supply Chain Management
- `ims.png` - Inventory Management
- `dms.png` - Document Management
- `quality.png` - Quality Management
- `compliance.png` - Compliance Management

## Placeholder
If a module thumbnail is missing, the system will display a puzzle piece icon as placeholder.
