# Platform Admin - Missing Pages Wireframes

> **Purpose:** Wireframe specifications for missing Platform Admin pages  
> **Target:** 50+ pages identified in gap analysis  
> **Priority:** High → Medium → Low

---

## Wireframe Standards

### Layout Structure
All pages use the Platform Admin layout:
```
┌─────────────────────────────────────────────────────────────────┐
│  [Logo]  Admin Portal          [Notifications] [User Menu]       │
├──────────┬──────────────────────────────────────────────────────┤
│          │                                                       │
│ Sidebar  │  Main Content Area                                    │
│  Menu    │                                                       │
│          │  ┌─────────────────────────────────────────────────┐ │
│ Dashboard│  │  Page Header                           [Actions] │ │
│ Tenants  │  ├─────────────────────────────────────────────────┤ │
│ Users    │  │                                                   │ │
│ Roles    │  │  Stats Cards (optional)                          │ │
│ Billing  │  │                                                   │ │
│ Notify   │  ├─────────────────────────────────────────────────┤ │
│ Files    │  │                                                   │ │
│ Logs     │  │  Filter Bar                                       │ │
│ Settings │  │  [Search] [Dropdown] [Dropdown] [Date]           │ │
│ DevTools │  │                                                   │ │
│ Analytics│  ├─────────────────────────────────────────────────┤ │
│ Integrate│  │                                                   │ │
│ Support  │  │  Data Table / Content                            │ │
│ Onboard  │  │                                                   │ │
│          │  │                                                   │ │
│          │  └─────────────────────────────────────────────────┘ │
│          │                                                       │
│          │  [Pagination]                                         │
└──────────┴──────────────────────────────────────────────────────┘
```

### Component Library
- **Table:** HeroUI Table with sortable columns
- **Cards:** HeroUI Card for stats/widgets
- **Forms:** HeroUI Input, Select, Textarea
- **Buttons:** HeroUI Button with variants
- **Modals:** HeroUI Modal for forms
- **Icons:** @heroicons/react/24/outline
