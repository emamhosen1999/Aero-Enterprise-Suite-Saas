## [1.0.0] - 2025-12-07

### Added
- Initial release of HRM module as standalone package
- Employee management with complete lifecycle
- Department management with hierarchical support
- Designation management with level support
- Multi-tenancy awareness (standalone, platform, tenant modes)
- Smart service provider with environment detection
- Comprehensive configuration options
- Feature flags for attendance, payroll, leave modules
- Employee ID generation system
- Leave management with multiple leave types
- Attendance tracking configuration
- Payroll integration settings
- Frontend components and pages (Inertia.js/React)
- API endpoints for all HRM resources
- Authorization policies
- Form request validators
- Database migrations for all HRM tables
- Comprehensive documentation
- Test suite structure

### Features
- 👥 Employee Management - Complete employee lifecycle
- 🏢 Department Management - Hierarchical organization
- 📊 Designation Management - Job titles and levels
- ⏰ Attendance Tracking - Time and attendance
- 🏖️ Leave Management - Leave requests and approvals
- 💰 Payroll Integration - Salary structures
