Aero HRM — Human Resource Management (Laravel 11 + Inertia.js v2)

Modern HR module built with Laravel 11, Inertia.js v2 + React 18, Tailwind CSS v4, and HeroUI. Ships in Standalone or SaaS (multi-tenant) mode.

Highlights

  • Employees, Departments, Designations
  • Leave Types, Requests, Approvals, Bulk Leave
  • Timesheets & Attendance (filters, search, pagination)
  • Role-Based Access Control (RBAC)
  • Polished UI with Dark Mode and consistent modals/forms/toasts

Tech Stack

  • Laravel 11 (PHP 8.2+)
  • Inertia.js v2 + React 18
  • Tailwind CSS v4 + HeroUI
  • MySQL/MariaDB

Modes

  • Standalone: Single-tenant HR system
  • SaaS: Multi-tenant, subdomain-based isolation

Screenshots

Requirements

  • PHP 8.2+, Composer 2+
  • Node.js 18+
  • MySQL 8+ or MariaDB 10.5+
  • Nginx/Apache configured to serve public/

Installation

Concise steps are included in the documentation: see Installation.md. Summary: configure .env, run composer install, npm ci && npm run build, php artisan migrate. For SaaS, also run tenant:create and tenant:migrate.
