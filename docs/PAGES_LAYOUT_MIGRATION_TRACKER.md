# Pages Layout Migration Tracker

This tracker lists all page-related files under `packages/*/resources/js/Pages/` and is used to migrate them to the layout contract (`StandardPageLayout`).

## Layout Contract Checklist (per page)
- [ ] Wrap page with `StandardPageLayout`
- [ ] Put action buttons in `actions` slot
- [ ] Put stats in `stats` slot (use existing `StatsCards` shape)
- [ ] Put filters/search in `filters` slot
- [ ] Put main table/grid in `children`
- [ ] Put pagination in `pagination` slot (if any)
- [ ] Replace page-local `getThemeRadius` with `useThemeRadius()` where applicable
- [ ] Remove page-local `isMobile/isTablet` resize listeners if `StandardPageLayout` covers it
- [ ] Remove duplicate Card/Header wrappers now owned by `StandardPageLayout`
- [ ] `get_errors` shows no new issues
